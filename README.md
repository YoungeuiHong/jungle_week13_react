## 👀 About

정글 13주차 나만의 무기 준비하기 과제로 구현한 React 애플리케이션입니다.

### ▪️ 기술 스택

* NextJS
* NextAuth.js
* Jotai
* Firebase

### ▪️ 구현 기능
* OAuth 로그인
* 게시글 CRUD 기능
* 반응형 웹 사이트
